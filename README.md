# Repository for my personal Blog


The main blog directory needs to have the most files. Otherwise, the postings will not show.

## How to start hugo

### Install hugo
    $ brew install hugo

### Install submodules

	$ git submodule update --init --recursive
 

### Build locally
	$ hugo server
